Bem-vindo ao Sistema de Manipulação de PDFs

Este sistema oferece diversas funcionalidades para trabalhar com arquivos PDF e imagens. Abaixo você encontrará um guia completo de como usar cada recurso.

1. Inserir Imagem

Insira uma página de imagem em um PDF existente, com total controle sobre a posição e tamanho.

Como usar:

Passo 1: Selecione o arquivo PDF base clicando na área de upload ou arrastando o arquivo.
Passo 2: Selecione a imagem (PNG, JPEG, GIF ou WEBP) que deseja inserir.
Passo 3: Visualize a imagem na seção de preview para confirmar.
Passo 4: Configure a posição (início, final ou específica) e o tamanho da página.
Passo 5: Clique em "PROCESSAR PDF" e o arquivo será baixado automaticamente.

2. Inserir PDF

Insira um PDF completo dentro de outro PDF em qualquer posição desejada.

Como usar:

Passo 1: Selecione o PDF principal (onde o outro será inserido).
Passo 2: Selecione o PDF que deseja inserir.
Passo 3: Escolha onde inserir: no início, no final ou depois de uma página específica.
Passo 4: Clique em "PROCESSAR PDF" para criar o documento combinado.
Dica: Use "Depois da página X" para inserir o PDF em uma posição específica do documento!

3. Processamento em Lote

Processe múltiplos PDFs de uma só vez, inserindo a mesma imagem em todos eles.

Como usar:

Passo 1: Selecione a imagem que será inserida em todos os PDFs.
Passo 2: Configure a posição e tamanho da página (será aplicado a todos).
Passo 3: Arraste ou selecione múltiplos arquivos PDF.
Passo 4: Clique em "PROCESSAR LOTE" e aguarde.
Passo 5: Os arquivos processados serão baixados automaticamente.
Importante: Você pode remover arquivos da lista clicando no X ao lado de cada item antes de processar.

4. Juntar PDFs

Combine múltiplos arquivos PDF em um único documento.

Como usar:

Passo 1: Selecione os arquivos PDF na ordem desejada.
Passo 2: Use o botão "Reordenar" ou as setas para mudar a ordem.
Passo 3: Digite o nome para o arquivo final (sem a extensão .pdf).
Passo 4: Clique em "JUNTAR PDFs" para criar o documento unificado.

Perguntas Frequentes

Qual o tamanho máximo dos arquivos?

  • PDFs: 50MB por arquivo
  • Imagens: 10MB por arquivo

Os arquivos são enviados para algum servidor?

Não! Todo o processamento é feito localmente no seu navegador. Seus arquivos não saem do seu computador.

Posso usar em dispositivos móveis?

Sim! O sistema funciona em navegadores modernos de computadores, tablets e smartphones.

Como funciona a inserção em posição específica?

Na inserção de imagem, ela é colocada ANTES da página indicada. Na inserção de PDF, ele é colocado DEPOIS da página indicada.

Atalhos e Dicas

  • Seções recolhíveis: Clique nos cabeçalhos azuis para mostrar/ocultar seções.
  • Drag & Drop: Arraste arquivos diretamente para as áreas de upload.
  • Múltiplos arquivos: Use Ctrl (Windows) ou Cmd (Mac) para selecionar vários arquivos de uma vez.
  • Limpar formulário: Use o botão "LIMPAR" para resetar todos os campos de uma vez.
